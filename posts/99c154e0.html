CODING初探

发表于 2021-01-29 分类于 Serverless
CODING 与腾讯云 Serverless 配合使用，可以一键部署网站应用

今天接收到 TencentServerless 公众号推送，文章介绍 CODING 与腾讯云 Serverless 配合使用，可以一键部署网站应用,我决定试一试。

CODING 静态网站服务是腾讯云 Serverless 团队联合 CODING 为开发者提供的便捷、稳定、高拓展性的静态网站资源托管服务，结合了 Tencent Serverless Framework 完整、高效的部署流程和 CODING 强劲的 Jenkins 构建队列，支持 Jekyll、Hexo 等多种部署框架。您无需自建服务器，即可一键部署网站应用，将静态网站分发至全网节点。

在腾讯云中搜索 CODING

点击 CODING DevOps 进入 CODING 的主界面

点击立即使用，进入工作台界面。

点击项目进入创建项目界面。

点击创建项目，选择 DevOps 项目

填写项目基本信息

完成后，会进入 CODING DevOps 项目引导界面

然后点击新建代码仓库

这里我选择的导入外部仓库。我将 git 上的 hexo 项目导了过来。点击确认后，会进入新的代码仓库界面

点击持续部署－〉静态网站

点击新建网站

填写好相关信息，点击确定，等待部署完成。

点击访问，就可以访问你的网站了。

点击小箭头，可以去配置自己的域名
